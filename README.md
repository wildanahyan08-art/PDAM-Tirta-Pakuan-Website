# PDAM Tirta Pakuan — Sistem Informasi Pelanggan

> Sistem manajemen pelanggan dan tagihan air PDAM berbasis web modern.

## ✨ Fitur

- **Landing Page** — Informasi publik, layanan, testimoni, FAQ
- **Autentikasi** — Login/logout dengan role-based redirect (Admin & Customer)
- **Admin Dashboard** — Kelola pelanggan, tagihan, dan layanan
- **Manajemen Pelanggan** — CRUD customer, filter & pencarian
- **Manajemen Tagihan** — Buat, edit, verifikasi pembayaran tagihan
- **Manajemen Layanan** — Kelola tarif dan jenis layanan air
- **Panel Pelanggan** — Lihat profil, tagihan, dan riwayat pembayaran
- **Responsive** — Tampilan optimal di desktop & mobile

## 🛠️ Tech Stack

| Teknologi | Keterangan |
|---|---|
| **Next.js 16** (App Router) | Framework React modern |
| **React 19** | Library UI |
| **TypeScript** | Type safety |
| **Tailwind CSS v4** | Utility-first styling |
| **shadcn/ui** | Komponen UI (Sidebar, Dialog) |
| **Framer Motion** | Animasi |
| **Tabler Icons + Lucide React** | Ikon |
| **React Toastify** | Notifikasi |
